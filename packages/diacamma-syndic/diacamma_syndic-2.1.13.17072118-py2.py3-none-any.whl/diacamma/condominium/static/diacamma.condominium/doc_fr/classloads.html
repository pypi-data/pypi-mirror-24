Les catégories de charges¶

Depuis le menu Copropriété/Les catégories de charges, vous pouvez définir des ensembles de lots où se répartisse différement des dépenses suivant chaque copropriétaires. Par exemple, vous pouvez distinguer en catégories de charges: les montés d’excalier, les garages, le chauffage, l’eau, ...
