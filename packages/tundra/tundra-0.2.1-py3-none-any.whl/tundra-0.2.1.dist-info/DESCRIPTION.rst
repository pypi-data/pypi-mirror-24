
Tundra
======

Pure Python, no dependencies Graph Algorithms module

Installing
----------

``pip install tundra``

Overview
--------

Structures
~~~~~~~~~~

-  Graph class
-  Digraph class

Algorithms
~~~~~~~~~~

Search
^^^^^^

-  Depth-first search
-  Breadth-first search

Spanning tree
^^^^^^^^^^^^^

-  Kruskal's Algorithm
-  Prim's Algorithm

Path
^^^^

-  Dijskra's Algoritm
-  Floyd-Warshall Algoritm
-  Nearest-neighbors hamiltonian cycle

Miscellaneous
^^^^^^^^^^^^^

-  Fringe
-  Greedy coloring
-  Proprety tests (is\_tree, is\_complete,
   ...)

Utilities
~~~~~~~~~

-  DOT language conversion
-  Export Graph to PNG
