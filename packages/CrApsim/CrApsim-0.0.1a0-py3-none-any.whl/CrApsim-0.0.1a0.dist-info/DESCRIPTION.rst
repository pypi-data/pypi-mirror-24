CrApsim
===============================

version number: 0.0.1
author: Torsten Zielke

Overview
--------

Creates Apsim-Project files from given parameters and files.

Installation / Usage
--------------------

To install use pip:

    $ pip install CrApsim


Or clone the repo:

    $ git clone TBD
