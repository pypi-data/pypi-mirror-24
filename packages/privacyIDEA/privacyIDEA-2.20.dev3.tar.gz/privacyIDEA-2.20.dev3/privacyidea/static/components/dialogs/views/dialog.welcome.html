Welcome

First, we would like to give you some background information on privacyIDEA.

privacyIDEA is an Enterprise Level Open Source Multi Factor Authentication System. You can use privacyIDEA with up to 50 assigned tokens without subscription. For more than 50 tokens you need a valid subscription with support and service level agreement.

You can find the complete source code of privacyIDEA at Github.

If you find a bug or if you want to contribute to privacyIDEA, please open an issue or pull request there.

The Community

If you have any question regarding privacyIDEA, please visit our community forum. Your question might already been answered there. You are welcome to file any new question or discuss any new idea there.

With you starting to use privacyIDEA, you become a part of this community. Every question you ask in the forum and every question you answer there helps others, to get started with privacyIDEA.

The Policies

If you do not want to this information again, please create your first policy in the scope "webui". You can use the template hide_welcome. Such policy will prevent this information from being displayed.

Policies are the way in privacyIDEA to define many behaviours. You can read more about policies in the online documentation.
