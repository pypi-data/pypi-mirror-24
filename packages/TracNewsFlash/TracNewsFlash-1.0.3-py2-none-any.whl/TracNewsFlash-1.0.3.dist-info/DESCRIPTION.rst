Notes
=====
A simple macro the create a colored sidebar.

Configuration
=============

To enable the plugin::

  [components]
  newsflash.* = enabled

Example
=======
An example usage::

  {{{
  #!NewsFlash
  = Version 2.0 =
  Version 2.0 has been released. Check out the [wiki:Download] page.

  = New website =
  Our project has a new website. Welcome!
  }}}
